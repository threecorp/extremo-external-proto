BookingEvent

BookingEvent is one entry in a tenant's booking change feed: an append-only,

cursor-paginated stream of booking lifecycle changes. An external consumer

polls "what changed since my last cursor", and the same feed backs webhook

delivery. Internal audit detail (actor identity, raw field diffs) is not

exposed.

FieldTypeLabelDescription
cursor string

Opaque cursor for this event. Pass the response's next_cursor back as the request cursor to fetch events after this one.

type BookingEventType

What kind of change this event represents.

booking_id uint64

Opaque public identifier of the booking this event is about.

occurred_at google.protobuf.Timestamp

When the change occurred (UTC).

booking Booking

Snapshot of the booking after the change. Absent if the booking has since been hard-deleted (the event type still records what happened).

BookingEventType

BookingEventType is the public kind of a booking change in the event feed.

Internal mutation kinds are folded into this stable public vocabulary (e.g.

reject/delete both surface as CANCELED).

NameNumberDescription
BOOKING_EVENT_TYPE_UNSPECIFIED 0

Unspecified / unknown event type.
BOOKING_EVENT_TYPE_CREATED 1

A booking was created.
BOOKING_EVENT_TYPE_UPDATED 2

A booking's attributes changed.
BOOKING_EVENT_TYPE_CONFIRMED 3

A booking was confirmed (accepted by the tenant).
BOOKING_EVENT_TYPE_CANCELED 4

A booking was canceled (canceled, rejected, or deleted).
BOOKING_EVENT_TYPE_NO_SHOW 5

The customer did not show up.

BookingStatus

NameNumberDescription

BOOKING_STATUS_DRAFT 2

Draft — not yet confirmed.
BOOKING_STATUS_RESERVED 3

Reserved / confirmed.
BOOKING_STATUS_ORDERED 4

Ordered (checked out / fulfilled).
BOOKING_STATUS_NO_SHOW 5

The customer did not show up.

SlotStatus

SlotStatus is the availability state of a calendar slot.

NameNumberDescription
SLOT_STATUS_UNSPECIFIED 0

Unspecified / unknown status.
SLOT_STATUS_AVAILABLE 1

The slot is bookable.
SLOT_STATUS_BUSY 2

The slot is partially taken (some capacity used).
SLOT_STATUS_FULL 3

The slot is fully booked.

ListBookingEventsRequest

ListBookingEventsRequest pages a tenant's booking change feed forward from an

opaque cursor.

FieldTypeLabelDescription
tenant_id uint64

Tenant whose event feed to read (must match the API key's tenant).

cursor string

Opaque cursor from a previous response's next_cursor. Empty starts from the beginning of the feed.

page_size int32

Page size (1..100).

ListBookingEventsResponse

ListBookingEventsResponse is a page of booking change events.

FieldTypeLabelDescription
events extremo.msg.external.v1.BookingEvent repeated

The events on this page, oldest-cursor first.

next_cursor string

Cursor to pass as the next request's cursor to continue after this page.

has_more bool

True if more events may exist after this page.

BookEventService

BookEventService streams a tenant's booking lifecycle changes as a

cursor-paginated feed. Read-only.

Method NameRequest TypeResponse TypeDescription
ListBookingEvents ListBookingEventsRequest ListBookingEventsResponse

ListBookingEvents returns booking change events after the given cursor, in ascending (oldest-first) order, so a consumer can advance through the feed incrementally by passing next_cursor back on each call.

Methods with HTTP bindings

Method Name Method Pattern Body
ListBookingEvents GET /api/external/v1/{tenant_id}/booking-events

end_at google.protobuf.Timestamp

1-based page number.

page_size int64

FieldTypeLabelDescription
